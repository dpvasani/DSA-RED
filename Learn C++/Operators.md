## üßÆ 1. Arithmetic Operators

**Purpose:** Perform basic mathematical operations.

| Operator | Description    | Example        | Analogy                            |
| -------- | -------------- | -------------- | ---------------------------------- |
| `+`      | Addition       | `a + b`        | Combining two quantities           |
| `-`      | Subtraction    | `a - b`        | Removing one quantity from another |
| `*`      | Multiplication | `a * b`        | Scaling a quantity                 |
| `/`      | Division       | `a / b`        | Splitting into equal parts         |
| `%`      | Modulus        | `a % b`        | Finding the remainder              |
| `++`     | Increment      | `++a` or `a++` | Increasing by one                  |
| `--`     | Decrement      | `--a` or `a--` | Decreasing by one                  |

**Example:**

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 3;
    cout << "Addition: " << a + b << endl;
    cout << "Subtraction: " << a - b << endl;
    cout << "Multiplication: " << a * b << endl;
    cout << "Division: " << a / b << endl;
    cout << "Modulus: " << a % b << endl;
    return 0;
}
```

**Output:**

```
Addition: 13
Subtraction: 7
Multiplication: 30
Division: 3
Modulus: 1
```

## üß† 5. Bitwise Operators

**Purpose:** Perform operations at the bit level.

| Operator | Description | Example    | Result                          |     |                           |
| -------- | ----------- | ---------- | ------------------------------- | --- | ------------------------- |
| `&`      | Bitwise AND | `a & b`    | Bits set in both a and b        |     |                           |
| \`       | \`          | Bitwise OR | \`a                             | b\` | Bits set in either a or b |
| `^`      | Bitwise XOR | `a ^ b`    | Bits set in a or b but not both |     |                           |
| `~`      | Bitwise NOT | `~a`       | Inverts bits of a               |     |                           |
| `<<`     | Left shift  | `a << 1`   | Shifts bits of a left by 1      |     |                           |
| `>>`     | Right shift | `a >> 1`   | Shifts bits of a right by 1     |     |                           |

**Example:**

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5; // Binary: 0101
    int b = 9; // Binary: 1001
    cout << "a & b: " << (a & b) << endl;
    cout << "a | b: " << (a | b) << endl;
    cout << "a ^ b: " << (a ^ b) << endl;
    cout << "~a: " << (~a) << endl;
    cout << "b << 1: " << (b << 1) << endl;
    cout << "b >> 1: " << (b >> 1) << endl;
    return 0;
}
```

**Output:**

```
a & b: 1
a | b: 13
a ^ b: 12
~a: -6
b << 1: 18
b >> 1: 4
```

## üß†Bitwise Operators in C++

### üß© Purpose:

They **manipulate individual bits** within integers ‚Äî often used for performance optimization, system programming, cryptography, etc.

---

### üîß Operator Summary:

| Operator | Description | Example  | Meaning                                       |                             |
| -------- | ----------- | -------- | --------------------------------------------- | --------------------------- |
| `&`      | Bitwise AND | `a & b`  | Only 1 where **both** bits are 1              |                             |
| `\|`     | Bitwise OR  | \`a      | b\`                                           | 1 where **either** bit is 1 |
| `^`      | Bitwise XOR | `a ^ b`  | 1 where bits are **different**                |                             |
| `~`      | Bitwise NOT | `~a`     | Inverts each bit (1 ‚ûù 0, 0 ‚ûù 1)               |                             |
| `<<`     | Left Shift  | `a << 1` | Shifts bits **left**, adds 0 on right         |                             |
| `>>`     | Right Shift | `a >> 1` | Shifts bits **right**, discards rightmost bit |                             |

---

### üîç Example with Binary Representation:

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5;  // Binary:  0101
    int b = 9;  // Binary:  1001

    cout << "a & b: " << (a & b) << endl;   // AND
    cout << "a | b: " << (a | b) << endl;   // OR
    cout << "a ^ b: " << (a ^ b) << endl;   // XOR
    cout << "~a: " << (~a) << endl;         // NOT
    cout << "b << 1: " << (b << 1) << endl; // Left Shift
    cout << "b >> 1: " << (b >> 1) << endl; // Right Shift

    return 0;
}
```

---

### üßÆ Step-by-step Binary Explanation:

#### Variables:

```
a = 5     ‚ûù 0000 0101
b = 9     ‚ûù 0000 1001
```

---

### ‚úÖ Outputs with Binary Logic:

| Expression | Binary Operation                  | Decimal Result                     | Explanation                                             |                             |
| ---------- | --------------------------------- | ---------------------------------- | ------------------------------------------------------- | --------------------------- |
| `a & b`    | 0000 0101 & 0000 1001 = 0000 0001 | `1`                                | Only bit 0 is set in both                               |                             |
| \`a        | b\`                               | 0000 0101 \| 0000 1001 = 0000 1101 | `13`                                                    | Combines set bits from both |
| `a ^ b`    | 0000 0101 ^ 0000 1001 = 0000 1100 | `12`                               | Only bits that are different                            |                             |
| `~a`       | \~0000 0101 = 1111 1010           | `-6`                               | 2‚Äôs complement of 6 (inverts bits)                      |                             |
| `b << 1`   | 0000 1001 << 1 = 0001 0010        | `18`                               | Left shift: multiply by 2 (adds 0 on right)             |                             |
| `b >> 1`   | 0000 1001 >> 1 = 0000 0100        | `4`                                | Right shift: divides by 2 (drops least significant bit) |                             |

---

### üí° Important Notes:

* **Negative Results:** Bitwise NOT (`~`) returns negative because of how signed integers are stored (2's complement).
* **Left/Right Shifting**:

  * `x << n` ‚Üí `x * 2^n`
  * `x >> n` ‚Üí `x / 2^n` (floor division)

## ‚úÖ Extract Digits in Original Order (Left to Right)

## üîÅ Reverse a Number in C++

### üß† Idea:

Take the last digit ‚Üí append it to the result ‚Üí remove it from the original ‚Üí repeat until 0.

---

### ‚úÖ Code Example:

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 123;
    int reversed = 0;

    while (a > 0) {
        int digit = a % 10;           // üîç Get last digit
        reversed = reversed * 10 + digit; // ‚¨ÖÔ∏è Shift digits left & add new digit
        a = a / 10;                   // üîÑ Remove last digit
    }

    cout << "Reversed number: " << reversed << endl;
    return 0;
}
```

---

### üì§ Output:

```
Reversed number: 321
```

---

### üßä Analogy:

Imagine you‚Äôre stacking blocks numbered `1`, `2`, and `3` vertically üß±üß±üß±. Reversing means lifting the last one first and stacking it into a new tower from the base. Now it becomes `3`, `2`, `1` üèóÔ∏è.

---

### üí° Bonus: Handle Reversed Number as a String

If you're okay treating numbers as text (e.g., for display), here's a shortcut using string:

## üîÅ Reverse an Integer Safely in C++

### üß† Objective:

* Reverse digits of an integer
* Support **negative numbers**
* Avoid **integer overflow**

---

### ‚úÖ Final Code with Comments:

```cpp
#include <iostream>
#include <climits> // For INT_MAX and INT_MIN
using namespace std;

int reverseInteger(int x) {
    int reversed = 0;

    while (x != 0) {
        int digit = x % 10;  // üîç Extract last digit

        // üõë Check overflow/underflow before adding digit
        if (reversed > INT_MAX / 10 || (reversed == INT_MAX / 10 && digit > 7))
            return 0; // Overflow
        if (reversed < INT_MIN / 10 || (reversed == INT_MIN / 10 && digit < -8))
            return 0; // Underflow

        reversed = reversed * 10 + digit; // ‚¨ÖÔ∏è Shift and add digit
        x = x / 10;                       // üîÑ Remove last digit
    }

    return reversed;
}

int main() {
    int number;
    cout << "Enter a number to reverse: ";
    cin >> number;

    int result = reverseInteger(number);

    if (result == 0 && number != 0) {
        cout << "‚ùó Overflow/Underflow occurred!" << endl;
    } else {
        cout << "‚úÖ Reversed number: " << result << endl;
    }

    return 0;
}
```

---

### üì§ Example Outputs:

#### ‚úÖ Case 1: Positive input

```
Input: 123
Output: ‚úÖ Reversed number: 321
```

#### ‚úÖ Case 2: Negative input

```
Input: -456
Output: ‚úÖ Reversed number: -654
```

#### üö´ Case 3: Overflow case (e.g., 1534236469)

```
Input: 1534236469
Output: ‚ùó Overflow/Underflow occurred!
```

---

### üìö Why Check for Overflow?

In C++, the range for `int` is:

* üîº `INT_MAX = 2147483647`
* üîΩ `INT_MIN = -2147483648`

To be safe before:

```cpp
reversed = reversed * 10 + digit;
```

We **predict** whether multiplying by 10 and adding will overflow.

---

### üßä Analogy:

Imagine a jar üì¶ can hold up to `2147483647` candies. Every digit you add is like tossing in more. Before you toss, make sure the jar won‚Äôt **overflow** üç¨üß®.

## ‚öôÔ∏è Division Operator (`/`) in Different Data Types

### üìå General Syntax:

```cpp
result = a / b;
```

The result **depends on the types** of `a` and `b`.

---

## üß™ Type-wise Behavior

| Type of `a` | Type of `b` | Result Type | Behavior                             |
| ----------- | ----------- | ----------- | ------------------------------------ |
| `int`       | `int`       | `int`       | Integer division (truncates decimal) |
| `float`     | `float`     | `float`     | Floating point division              |
| `int`       | `float`     | `float`     | Integer promoted to float            |
| `double`    | `int`       | `double`    | Integer promoted to double           |
| `char`      | `int`       | `int`       | `char` promoted to ASCII int         |

---

## üî¢ Examples with Outputs

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 3;
    float x = 10.0f, y = 3.0f;
    double d = 10.0;
    char ch = 'A'; // ASCII 65

    cout << "üîπ int / int = " << a / b << endl;         // 3
    cout << "üîπ float / float = " << x / y << endl;     // 3.33333
    cout << "üîπ int / float = " << a / y << endl;       // 3.33333
    cout << "üîπ double / int = " << d / b << endl;      // 3.33333
    cout << "üîπ char / int = " << ch / b << endl;       // 65 / 3 = 21
    return 0;
}
```

### üßæ Output:

```
üîπ int / int = 3
üîπ float / float = 3.33333
üîπ int / float = 3.33333
üîπ double / int = 3.33333
üîπ char / int = 21
```

## ‚ö†Ô∏è Important Notes

### ‚ùó 1. **Integer Division Truncates**

```cpp

cout << a / b;  // Output: 3 (NOT 3.5)
```

üí° Analogy: Like dividing 7 apples üçé among 2 kids. Each kid gets **3 full apples**, and the half is discarded.

---

### ‚ö†Ô∏è 2. **To Get Decimal Results: Use float/double**

```cpp
float result = 7 / 2.0; // or cast one operand: (float)7 / 2;
```

‚úÖ Output: `3.5`

---

### ‚ö†Ô∏è 3. **Division by Zero**

* Integer division by zero: ‚ùå **runtime error**
* Floating-point division by zero: ‚úÖ **Infinity or NaN**

```cpp
int a = 5, b = 0;
float x = 5.0f, y = 0.0f;

// cout << a / b; // ‚ùå CRASH

cout << x / y; // ‚úÖ Output: inf
```
